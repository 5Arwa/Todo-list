 To Do List App (PHP)

A simple To Do List web app built with PHP, MySQL, and Bootstrap.  
The app includes form validation and allows users to easily manage their tasks.



 Features
- Add new to-do items  
- Edit and delete tasks  
- Validation for empty inputs  
- Change task status (All → Doing → Done)  
- Responsive design with Bootstrap  


 Technologies Used
- PHP  
- MySQL (PDO)  
- HTML / CSS / Bootstrap  
- JavaScript (for alert confirmation)


 How to Run the Project
1. Import the SQL database file into phpMyAdmin.  
2. Update the database connection in your PHP config file.  
3. Start Apache & MySQL from XAMPP.  
4. Open `index.php` in your browser.  



  
Simple To Do List project built for learning PHP and practicing CRUD operations.
